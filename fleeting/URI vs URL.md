### URI(Uniform Resource Identifier)

- 인터넷에서 특정 리소스를 식별하는 문자열의 일반적인 형식

### URL(Uniform Resource Locator)

- URI 의 하위 집합으로, 웹 상의 리소스 위치를 나타낸다.
- 리소스에 어떻게 접근할 수 있는지에 대한 정보를 포함하고 있다.s

일반적으로 URI는 URL을 포함하는 넓은 개념이다.
